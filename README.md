# ISEA Linux Server Environment Labs
## BRG-27 Labs

---

## Overview

This repository documents the practical lab activities completed for the ISEA module.

The labs demonstrate Linux server administration, command-line operations, automation, and service configuration.

---

## Lab Environment

Ubuntu Linux  
Oracle VirtualBox  
Linux Terminal  
GitHub

---

## Day 1 – Linux Setup and Command Line

- Installed Ubuntu Linux using VirtualBox
- Navigated directories using `pwd`, `ls`, `cd`
- Created files using `mkdir` and `touch`

---

## Day 2 – Services, Permissions and Automation

- Checked file permissions using `ls -l`
- Modified permissions using `chmod`
- Installed and verified SSH service
- Created Bash scripts for automation
- Scheduled tasks using cron

---

## Day 3 – DNS and Security

- Verified DNS records using `nslookup`
- Installed SSL certificates using Certbot
- Explored server security mechanisms

---

## Day 4 – Additional Server Service

- Installed Apache web server
- Verified service status using `systemctl`
